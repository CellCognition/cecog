Global Threshold

Segmentation using simple global thresholding. The threshold can be set statically or adapted iteratively.

Median radius

Smooting the image before segementation using a median filter (radius in pixels).
Note: Filtered image is used for segmentation, not for feature calculation. This means there is no influence on feature values, except the shape features. The parameter scales with lens magnification

Threshold

This parmenter is the static threshold. Intensitiy values greater than this value are labeled as foreground.

Static threshold

If this flag is set, the thresholding is performed only once i.e. segmentation is a simple static global threshold. If the flag is off the threshold is updated iterativeley according to the formula threshold_new = (mu_bg + mu_fg)/2, where mu are the mean intesities of foreground and background respectively. The iteration stops if the threshold does not change between two iteration (255 iterations max.). The parameter Threshold is used as start parameter.

Split & merge by shape

Optional correction of under-segmentation by a split and merge approach. Objects in close spatial proximity might not be separated (under-segmentation). Based on the shape information (distance transform) objects are split by a watershed algorithm, which often yields over-segmentation. Object candidates are probed for merging by their size and circularity (rounder objects above a certain size threshold are preferred). See Wählby et al. 2004.

Gauss radius

Size of the Gauss filter for smoothing the distance transformed image. This Parameter scales with lens magnification

Min. seed distance

Minimal allowed distance in pixel between objects (seed distance for watershed). This Parameter scales with lens magnification

Object size threshold

Minimal size in pixels of a split object. This parameter scales square with lens magnification.

Window size

Width of the moving average window in pixel (window is squared).
(parameter scales directly with lens magnification)

Object filter

Object filtering based on size (in pixel) and mean. intensity (8bit grey level values). A value set to -1 means that the specific bound will be ignored.

Min. object size

Remove all objects smaller than this parameter.

Max. object size

Remove all objects larger than this parameter.

Min. average intensity

Remove all ojects dimmer than this parameter.

Max. average intensity

Remove all objects brighter that this parameter
