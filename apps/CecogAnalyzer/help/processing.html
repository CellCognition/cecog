Processing

This section combines all previous processing steps into one work-flow. Individual steps or processing of the secondary channel can be enabled or disabled. A separate window allows selection of different image processing results.

Parameters

Primary channel

Processing steps of the primary channel

Classification

Enables classification as defined in Classification.

Tracking

Enables object tracking, see Tracking.

Event selection

Enables event selection, see Event selection. (This option requires classification and tracking.)

Error correction

Executes the error correction based on the found events, see Error Correction. (This option requires event selection.)

Secondary channel

Processing steps of the secondary channel

Classification

Enables classification as defined in Classification.

Error correction

Executes the error correction based on the found events, see Error Correction. (This option requires event selection.)

Control Panel

  • Start processing - The entire processing pipe-line as define above is started. Note that only positions and frames as defined in General will be processed.
