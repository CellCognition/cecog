Segmentation: Ilastik classifier

ilastik is a simple, user-friendly tool for interactive image pixel classification. This plug-in implements a threshold based segmentation based on the output of the ilastik classifier.

ilastik

In order to use the plug-in you have to install ilastik 0.5 from source as well as CellCognition from source (note that the ilastik 0.5 python source has to be in your PYTHONPATH).

First, you have to export the learned ilastik classifier into into an hdf5 file (.h5): after you created an ilastik project and learned your classes of interest, you can export the classifier by clicking the "Export classifier" button.

Using this plug-in

This plugi-in is basically the same as the adaptive threshold plug-in. The difference is that, the threshold is not based on the raw data, but uses the probability maps as generated by the ilastik classifier.
