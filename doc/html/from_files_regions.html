Segmentation: Load regions from file

This plug-in implements the loading of the segmentation from previously generated image files with a different software such as Fiji or CellProfiler

Match the corresponding segmentation to the raw images

To match every single raw image in CellCognition with the corresponding pre-computed segmentation, one has to provide a mapping by a regular expresion string. For example, the mapping 

        
          ^%(plate)s$/^%(pos)s$/.*P%(pos)s_T%(time)05d_C%(channel)s_Z%(zslice)d_S1.tif

matches the each raw image from

  • plate %(plate)=MyPlate
  • position %(pos)=A01_01
  • time frame %(time)=00002(which is a digit with five leading zeros)
  • channel name %(channel)= gfp
  • z-slice %(zslice)=1

to the file stored in path in 

      MyPlate/A01_01/whatever_PA01_01_T00002_Cgfp_Z1_S1.tif
